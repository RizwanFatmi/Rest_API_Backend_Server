

  ABOUT THE PROJECT
  -------------------------------------------------------------------------------------
  It is a backend server which performs CRUD operation using "RestAPI" architecture.
  In this project I used "Node.js, Express.js, MongoDb".



-----------------------------------------------------------------------------------------------------------------------------
                                     How to run the project
-----------------------------------------------------------------------------------------------------------------------------

 
  1. Download and extract the "RestAPI_Backend" file. After extracting the file you will get the "RestAPI_Backend" folder	

  2. Install the "Visual Studio Code" and open the "RestAPI_Backend" folder in your "Visual Studio Code".

  3. Open the terminal and write this comman "nodemon index.js". It will run the backend server.

  4. Now, use the postman or thunderclient for CRUD operations.



 Note: Make sure you have an internet connection and Node.js installed on your computer.
